# DI_UD05_PP01

Aplicacion de libreta de direcciones realizada con JavaFX, FXML, CSS y Maven.
El proyecto implementa las partes 1 a 4 del tutorial JavaFX de Code Makery.

## Requisitos

- JDK 25
- Maven 3.9 o superior

## Ejecutar

```powershell
mvn clean javafx:run
```

## Funciones

- Listado de personas mediante `TableView`.
- Visualizacion de los datos de la persona seleccionada.
- Creacion, edicion y eliminacion de personas.
- Dialogo modal y validacion de campos.
- Formato de fecha `dd.MM.yyyy`.
- Tema oscuro mediante CSS.
